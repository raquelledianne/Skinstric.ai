# Skinstric AI

A premium, AI-inspired skincare onboarding experience designed as a modern SaaS-style interface.

Skinstric simulates a personalized skincare intelligence platform through immersive UI motion, interactive navigation, and a luxury-grade visual system.

## 🚀 Live Demo
https://skinstric-alpha.vercel.app/

---

## ✨ Overview

Skinstric AI is a frontend experience that simulates a next-generation skincare SaaS product.  
It focuses on interactive motion design, guided user flow, and premium UI aesthetics.

The goal of this project is to demonstrate strong skills in:

- React component architecture
- State-driven UI interactions
- Camera + file upload workflows
- Animation design using CSS transforms
- Modern SaaS landing page UX patterns

---

## 🎨 Experience Highlights

### Motion-Driven UI
- Smooth directional hover animations
- Soft easing curves for premium feel
- Layered opacity effects for depth

### Luxury Design System
- Minimal, high-end skincare aesthetic
- Strong typography hierarchy
- Intentional whitespace and spacing
- Interaction-first visual design

### Performance-Focused Animation
- GPU-accelerated transform animations
- Avoids layout shift (no reflow animations)
- Lightweight React state management

## 🛠️ Tech Stack
